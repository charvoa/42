_Project developed by Nicolas Charvoz, Serge Heitzler, Antoine Garcia, Louis Audibert and Nicolas Girardot_

Le 42sh est un shell recodé en C.


Les fonctionnalités importantes du programme : 

* Parser LL (';', '&&', '||', '>>', '<<', '>', '<') 
* Built-in
* Alias
* History
